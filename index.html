⚠️ 通信遮断警告（セキュリティルール未設定）

Firebase Firestoreのデータベースアクセス権限がありません。このエラーを解決するには、以下の手順を行ってください：

  1. Firebase Console にアクセスし、プロジェクトを開く
  2. 構築メニューの Firestore Database ➔ 「ルール(Rules)」タブを開く
  3. ルール文全体を以下の内容に書き換え、「公開」ボタンを押す
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /comments/{document} { allow read, write: if true; } } }
